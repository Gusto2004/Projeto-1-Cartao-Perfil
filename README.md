# Cartão de Perfil

Página pessoal simples, com visual inspirado num editor de código, feita para praticar HTML e CSS puro (sem frameworks).

## Estrutura

- **Header** — logo/nome + navegação
- **Hero** — apresentação pessoal
- **Projetos** — cards com os projetos feitos até agora
- **Footer** — links de contacto

## Tecnologias

- HTML5
- CSS3 (Flexbox + Grid)

## Como correr

Abre o `index.html` no browser, ou usa a extensão "Live Server" no VSCode para veres as alterações em tempo real.
